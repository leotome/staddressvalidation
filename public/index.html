Introduction

The Address Validation Application Programming Interface (API) can provide direct automated integration in your application about correct and valid address information.

This API uses two inputs to validate an address: "Country" and "Postal Code". When both are provided, and if valid data is provided, the server will be able to return the full address in JSON format.

What does it does for you?

The Address Validation API is a valuable addition to any business. It allows to reduce the chance of bad data quality, and avoids incorrect information, which can interfere with the fluidity of your business. It's currently available to retrieve information about the following countries:

  • Brazil
  • Portugal

How to operate?

This API is REST-compliant, which means, it uses the HTTP verb "GET".
